# Personal Productivity Suite

This project is a command-line based Personal Productivity Suite developed using Python.  
It demonstrates core Python concepts such as Object-Oriented Programming, file handling,
error handling, and modular code design.

## Features
- Calculator
- Notes Manager (JSON-based persistence)
- Timer
- File Organizer
- Unit Converter
- Backup & Restore functionality

## Project Structure
- main.py: Entry point of the application
- modules/: Contains all functional modules
- Data/: Stores notes and backup files
- docs/: Screenshots and documentation

## How to Run
```bash
python main.py
